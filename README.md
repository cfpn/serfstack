## 一些使用小贴士

最后分享几个实用建议，能帮你更好地使用SerfStack的服务：

**第一，善用快照和备份功能。** VPS套餐包含每周自动备份到远程服务器，但只保留最近的备份。如果你要做重要的系统升级或配置变更，建议先手动创建快照，万一出问题可以快速回滚。备份存储100GB是免费的，需要更多空间可以加钱扩到1TB。

**第二，注意带宽限制。** VPS套餐的带宽虽然是1Gbps端口，但每月流量是有限制的（根据套餐不同）。如果你的网站流量突然暴增，可能会超出流量限额。独立服务器的100Mbps套餐是不限流量的，1Gbps套餐包含30TB/月，一般也够用。

**第三，合理配置DDoS防护。** 基础的L2-L4层DDoS防护是包含在所有套餐里的，但如果你需要更高级的防护（比如针对单个网站的应用层防护），需要额外付费。他们提供的方案包括地理位置封锁、免费SSL证书和域名DNS托管，起价97美元。对于容易被攻击的业务来说，这笔钱花得值。

**第四，控制面板选择。** 如果你不擅长命令行操作，建议加装ISPmanager或Plesk控制面板。ISPmanager便宜一些（3.5美元/月起），界面简洁，适合管理多个网站。Plesk功能更强大（14.5美元/月起），但也更复杂。两个面板都支持一键安装WordPress、Joomla等常用CMS系统。

**第五，SSL证书别忘了配置。** 所有套餐都包含免费的Let's Encrypt证书，但需要你自己去配置。如果觉得自动续期麻烦，可以买他们提供的Sectigo® Positive SSL证书，才1美元/年，而且不用操心续期问题。
